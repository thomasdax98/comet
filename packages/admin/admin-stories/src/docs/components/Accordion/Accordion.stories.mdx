# Accordion

> Accordion allows user to show or hide content sections.
>
> \- [MUI Docs](https://mui.com/material-ui/react-tooltip/)

The Comet Admin `Accordion` extends [Material UI's `Accordion`](https://mui.com/material-ui/react-accordion) by adding custom style and elements such as supportText, endAdornment and initialExpanded.

#### Props

| Name              | Type   | Description                                                                      |
| :---------------- | :----- | :------------------------------------------------------------------------------- |
| initialExpanded          | boolean (optional) | Defines Accordion initial state (default: false) |


In addition to its own props, the Accordion component also accepts all props from the Material UI Accordion component.
